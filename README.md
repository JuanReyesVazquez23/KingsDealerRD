KingsDealer

Proyecto en producción: https://kingsdealer.up.railway.app

Descripción del Proyecto

KingsDealer es una plataforma web de compra y venta de vehículos en República Dominicana.

La aplicación permite a los usuarios:

- Explorar vehículos disponibles.
- Buscar vehículos mediante filtros avanzados.
- Ver detalles completos de cada vehículo.
- Contactar vendedores.
- Solicitar financiamiento.
- Publicar vehículos para venta.
- Consultar ofertas destacadas.

El objetivo principal es ofrecer una experiencia moderna, rápida y confiable para compradores y vendedores de vehículos.

Funcionalidades Principales

Catálogo de Vehículos

- Listado de vehículos.
- Filtros por marca, modelo, año y precio.
- Visualización de detalles del vehículo.
- Galería de imágenes.

Financiamiento

- Formulario de solicitud.
- Gestión de solicitudes de crédito.
- Información sobre opciones de financiamiento.

Contacto

- Formularios de contacto.
- Integración con WhatsApp.
- Comunicación entre clientes y vendedores.

Publicación de Vehículos

- Creación de anuncios.
- Gestión de información del vehículo.
- Subida de imágenes.
- Edición y eliminación de publicaciones.

Panel de Administración

KingsDealer incluye un panel administrativo privado para la gestión completa de la plataforma.

Funciones del Administrador

- Crear vehículos nuevos.
- Editar vehículos existentes.
- Eliminar vehículos.
- Subir imágenes de vehículos.
- Gestionar inventario.
- Administrar publicaciones destacadas.
- Gestionar solicitudes de financiamiento.
- Supervisar el contenido de la plataforma.
- Actualizar información visible para los usuarios.

Reglas del Panel

- Solo usuarios con permisos de administrador pueden acceder.
- Todas las acciones deben validarse en el backend.
- Mantener registros de cambios importantes cuando sea posible.
- Priorizar la seguridad y protección de datos.
