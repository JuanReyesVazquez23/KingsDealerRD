## Descripción

**KingsDealerRD** es una aplicación web para un concesionario de vehículos desarrollada con **Flask (Python)**. El proyecto ofrece una experiencia moderna, rápida e intuitiva, permitiendo a los usuarios explorar el catálogo de vehículos de forma sencilla y eficiente.

La plataforma está inspirada en el diseño de un concesionario profesional e incorpora un **panel de administración oculto**, desde el cual es posible gestionar el inventario de vehículos y actualizar la información del sitio de manera segura.

---

## Características

- Catálogo de vehículos.
- Interfaz moderna, rápida e intuitiva.
- Diseño adaptable a dispositivos móviles y computadoras.
- Visualización detallada de cada vehículo.
- Apartado de financiamiento para clientes.
- Sección con información bancaria para facilitar los procesos de pago.
- Panel de administración oculto.
- Agregar, editar y eliminar vehículos.
- Gestión del contenido del sitio web.

---

## Tecnologías utilizadas

- Flask (Python)
- HTML5
- CSS3
- JavaScript
- PostgreSQL

## Instalación

### Clonar el repositorio

```bash
git clone https://github.com/JuanReyesVazquez23/KingsDealerRD.git
```

### Acceder al proyecto

```bash
cd KingsDealerRD
```

### Instalar las dependencias

```bash
pip install -r requirements.txt
```

### Ejecutar la aplicación

```bash
python app.py
```

## Objetivo del proyecto

El objetivo de este proyecto es aplicar conocimientos de desarrollo web mediante la creación de una aplicación completa para la gestión de un concesionario de vehículos, desarrollando un backend con **Flask (Python)**, una interfaz moderna y una base de datos **PostgreSQL** para administrar la información de forma eficiente.
